# 🏦 C Banking System

A console-based banking system written in C with file handling.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- File Storage (Persistent Data)

## 📂 Files
- `main.c` → Main source code
- `accounts.dat` → Stores account data
- `transactions.txt` → Stores transaction history

## 🛠️ How to Run

```bash
gcc main.c -o bank
./bank
```

## ⚙️ Technologies Used
- C Programming
- File Handling

## 📌 Notes
- Data is stored locally using files
- No database used
- Simple CLI-based system
